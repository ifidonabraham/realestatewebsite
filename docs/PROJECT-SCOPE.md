# Project Scope: Real Estate Finder MVP

## Goals
Develop a modern real estate platform that connects property seekers with verified agents and owners.

## MVP Framework (MoSCoW)

### 1. Must Have (Essential)
*   **Property Search:** Basic filters for location, price, and property type.
*   **Authentication:** User registration and login for Buyers and Agents.
*   **Property Details:** Single property view with images, description, and price.
*   **Direct Contact:** A way for buyers to message agents via a simple contact form.
*   **Mock Database:** A local `db.json` to store users, properties, and messages during development.

### 2. Should Have (Targeted for V1)
*   **Agent Dashboard:** Tools for agents to manage their own property listings.
*   **Real-time Chat:** A messaging interface for direct communication.
*   **Image Uploads:** Support for agents to upload property images to a storage bucket (e.g., Supabase Storage).
*   **Property Map:** Integration with Leaflet/Google Maps to show property locations.

### 3. Could Have (Future Enhancements)
*   **Wishlist/Favorites:** Users can save properties for later.
*   **Property Tours:** Integration for virtual tours or video walkthroughs.
*   **Advanced Analytics:** Dashboards showing property view statistics for agents.
*   **Verified Badges:** Identity verification for high-trust agents.

### 4. Won't Have (Out of Initial Scope)
*   **Payment Processing:** Direct transactions or escrow services.
*   **Mortgage Calculator:** Built-in financial tools.
